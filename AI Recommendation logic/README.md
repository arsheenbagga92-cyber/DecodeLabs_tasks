# AI Course Recommendation System

## Project Overview

This project demonstrates a simple AI Recommendation System using Natural Language Processing (NLP).

The system recommends courses based on user interests by comparing text similarity using TF-IDF Vectorization and Cosine Similarity.

## Technologies Used

- Python
- Scikit-Learn

## Features

- Takes user interests as input
- Uses TF-IDF Vectorizer
- Calculates Cosine Similarity
- Displays the top 5 recommended courses
