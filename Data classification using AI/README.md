# Iris Flower Classification Using AI

## Project Overview

This project demonstrates supervised machine learning using the Iris Dataset.

A K-Nearest Neighbors (KNN) classifier is trained to identify iris flower species based on:

- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

## Technologies Used

- Python
- Scikit-Learn
- Pandas
- Matplotlib
- Seaborn

## Workflow

1. Load Iris Dataset
2. Split Data into Training and Testing Sets
3. Scale Features
4. Train KNN Classifier
5. Generate Predictions
6. Evaluate Performance

## Results

The model achieves very high accuracy on the Iris Dataset.
